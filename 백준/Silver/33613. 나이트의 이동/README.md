### 출력

### 메모

우선 직접 그려보자!

5x5 배열에서 직접 그려서 시도해보면, 시작점 위치 기준으로 체스판 배열의 형태로 나이트의 위치가 정해지게 됨. 시작점 위치 R + C가 짝수면 (n**2)//2 + 1 만큼, 홀수면 (n**2)//2 만큼.

단, 이때 예외가 몇 가지 있는데,

1. 3x3 배열 중 시작 위치가 (2, 2)일때
   33배열에서, 나이트가 가운데 위치하게 되면, 움직일 수 없으므로 나이트는 연산을 진행하지 못한다. 따라서 위치할 수 있는 칸의 개수는 처음 시작 위치인 1이 나옴.

2. 4x4 배열처럼 짝수 배열에서는 R + C의 결과에 상관없이 동일한 개수가 나오게 된다.
   따라서, R + C가 짝수일 때 +1을 연산 이후가 아닌 먼저 더한 뒤에 //2를 수행하면 정상적으로 동작할 수 있음.
