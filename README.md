# 📚 Algorithm Study

이 저장소는 **백준(BOJ)과 프로그래머스(Programmers)의 알고리즘 문제 풀이 기록**을 정리한 공간입니다.  
풀이한 코드와 체감 난이도를 정리하며, **다양한 알고리즘 문제 해결 능력을 키우는 것**이 목표입니다!

## 🚀 사용 언어

- Python 🐍

## 🎯 난이도 표기 방식

개인적으로 느낀 체감 난이도를 문제 이름 옆에 표시합니다.

- ⭐ : 난이도 (개수가 늘어날수록 어려웠음)
  - ⭐ : 쉽게 해결
  - ⭐x5 : 매우 어려웠음
- ❣️ : (좀 더 쉽게 풀 수 있었던 문제)
  - 새로운 개념이 필요했던 문제
  - 더 간단한 풀이법이 있었던 문제
